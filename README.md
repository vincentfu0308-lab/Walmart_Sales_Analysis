# Walmart Sales Analysis

## Project Overview

This project analyzes Walmart historical sales data using Python, MySQL, SQL, and data visualization. The goal is to build an end-to-end data pipeline that extracts raw sales data, cleans and transforms it into an analysis-ready dataset, loads it into MySQL, and generates business insights about store performance, holiday sales impact, weekly sales trends, and store type differences.

The project demonstrates practical data analytics skills including ETL pipeline design, data cleaning, SQL database loading, validation queries, reusable SQL views, and Python-based exploratory data analysis.

## Business Questions

This project focuses on answering the following questions:

- Which Walmart stores generated the highest total sales?
- How do holiday weeks compare with non-holiday weeks in average weekly sales?
- What is the overall weekly sales trend over time?
- Which store types contribute the most sales?
- How can cleaned sales data be structured for future BI reporting or forecasting?
