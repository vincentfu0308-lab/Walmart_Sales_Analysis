# Brazilian E-Commerce Delivery Analytics — Olist

## Project Overview

This project analyzes the Brazilian E-Commerce Public Dataset by Olist and builds an end-to-end data science workflow to understand and predict delivery performance.

The main goals are:

1. Predict actual delivery days for each order.
2. Predict whether an order will be delivered late.
3. Generate model evaluation reports and visualizations in Jupyter Notebook.

The project follows this pipeline:

```text
ETL → EDA → Feature Engineering → Regression Modeling → Classification Modeling → Reporting
